Instruções para execução do código:

-> Abra o terminal, em seguida vá até a pasta.
-> Para executar o algoritmo, você deve executar com ./<nomedoprograma> <instancia>, no nosso caso será:
	./nneighbor <instancia>

Obs.: Essa instancia poderá ser qualquer uma do tipo TSP com as distancias euclideanas em 2D.
O programa criará um arquivo do tipo texto(saida.txt) contendo o grafo de melhor caminho e o custo.
